# Overview

Alexandrite Software Libraries for JavaScript (asljs) provides essential
utilities and functions to enhance everyday JavaScript development.

Libraries:

- [eventful](https://github.com/alex-netkachov/asljs-eventful) - adds
  on/off/emit to any object.
- [inspector](https://github.com/alex-netkachov/asljs-inspector) -
  comprehensive workflow inspector for tracking instances, events, and errors.
- [observable](https://github.com/alex-netkachov/asljs-observable) - makes
  any object emit events on property changes.
- [machine](https://github.com/alex-netkachov/asljs-machine) - provides
  a state-machine framework for organizing control flow.
- [money](https://github.com/alex-netkachov/asljs-money) - provides
  utilities for handling monetary values.
