# Data Format

Describes the format of a data file format, which stores particular entities in
the project.

## Properties

- Id - A unique identifier of the data file format, in the form of `DF<number>`.
- Name - A name of the data file format, filename without ID and extension.

## Location

- Files: `../development/**/DF*.md`

## Rules

- RL10 - At least one test file has data file format ID in its content.
