## Rules

### RL1

Article start with a level 1 heading, which is the file name without
extension. Exceptions: when the file name is all caps (e.g., `README.md`).

### RL2

Links to local resources must point to existing locations (files or
directories). Links that are longer than 20 characters must be a reference
link.
